# Discord post — ShatterApp update v2.15.1

**ShatterApp v2.15.1 — Custom builder card fixes**

Small patch just landed on [shatterapp.com](https://shatterapp.com):

**🃏 Custom Builder fixes**

- **Abilities card font scaling** — the card now automatically shrinks font and icon sizes as ability text grows, across four tiers. Long ability blocks used to get clipped off the bottom of the card — they should now always fit
- **Card previews on desktop** — the phase 1–4 card previews were rendering at full canvas size on desktop and overlapping surrounding content. Fixed — they now scale to their containers properly

No data changes, no unit changes. Just builder UX.
