ShatterApp — free fan-made companion app (list builder, unit browser, struggle tracker)

## Post Body

Hi all,

I've put together a free web companion app for Shatterpoint and wanted to share it with the community here.

**ShatterApp** — [shatterapp.com](https://shatterapp.com)

It runs in the browser, no account or installation required. It's also installable as a PWA if you prefer a home-screen shortcut.

**What's in it:**

*Browse* — a searchable, filterable grid of all characters. Filters cover unit type (Primary/Secondary/Support), era, tags, and pack. Tapping a card opens the full profile: card art with flip animation to show the back, stance images, all stats, and a keyword glossary for any rules text. There's also a side-by-side unit comparison tool.

*Build* — a Strike Force builder for both squads. It validates points in real time, lets you assign a mission, toggle Premiere format, name your list, and share it via a URL that recipients can import directly.

*Play* — a struggle tracker intended for use during the game. Beats trying to keep a mental count or scribbling on a notepad.

*Collection* — a simple pack ownership tracker with per-era progress indicators.

Data sourced from pointbreaksw.com. Code is open source on GitHub if anyone's interested.

Happy to add things — let me know if there's something that would be genuinely useful.
